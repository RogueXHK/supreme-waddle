Como Usar

Passo a passo para converter sua planilha

1

Baixe o Modelo

Na aba "Planilha Modelo", baixe o arquivo .xlsx com as colunas corretas e exemplo

2

Preencha os Dados

Preencha a planilha a partir da linha 4 (linha 2 = descrições, linha 3 = exemplo)

3

Escolha o Modo

API POST/PUT para novos endpoints (ProdutoIntegracaoRequestDTO), Lote POST/PUT para upload no portal (depreciado), Completo para formato de exportação

4

Converta

Faça upload da planilha, clique em converter e baixe o JSON gerado

5

Use na API

Envie o JSON para POST /catp/api/ext/produto/{cpfCnpjRaiz} ou PUT /catp/api/ext/produto/{cnpj}/{codigo}

6

Importe do Portal

Use "JSON → Excel" para converter um JSON exportado do portal em planilha editável

Campos Obrigatórios

Campos que devem estar preenchidos na planilha

Campo Descrição Regra
denominacao Nome do produto Máximo 120 caracteres
descricao Descrição detalhada Máximo 2000 caracteres
cpfCnpjRaiz CNPJ raiz da empresa 8 dígitos, somente números
modalidade Tipo de operação IMPORTACAO ou EXPORTACAO
ncm Classificação fiscal Exatamente 8 dígitos numéricos

📋 Como montar a planilha

A planilha deve ter pelo menos 2 colunas. O sistema identifica automaticamente pelos nomes do cabeçalho:

Coluna Nomes aceitos Descrição
codigo codigo, código, code, produto, cod, codigoProduto ✅ Código do produto no catálogo (número retornado pelo Siscomex)
codigoOperadorEstrangeiro codigoOperadorEstrangeiro, operador, codigo operador, operador estrangeiro ✅ Código do operador estrangeiro cadastrado no Siscomex
codigoPais codigoPais, codigo pais, país, pais, country ✅ Código do país do operador (ou preencha no campo acima)

Como funciona: O sistema gera um JSON no formato PUT do portal. Cada linha da planilha gera um produto com o campo codigoOperadorEstrangeiro preenchido. Use o JSON gerado para fazer upload no portal (modo PUT) ou via API PUT /ext/produto/{'{'}cpfCnpjRaiz{'}'}/{'{'}codigo{'}'}.

⚠️ Importante: O JSON gerado contém apenas os campos de vinculação. Para enviar via portal, é necessário usar junto com os dados completos do produto (denominacao, descricao, NCM, atributos, etc.).
